# GA-Breast-Cancer-Prediction
"An Explainable Breast Cancer Prediction Framework Using Genetic Algorithm-Based Feature Selection and Support Vector Machine"
This project proposes a Genetic Algorithm (GA)-optimized SVM framework for breast cancer prediction using the Wisconsin Breast Cancer Dataset.

Features:
- Genetic Algorithm Feature Selection
- SVM Classification
- Cross Validation
- ROC-AUC Analysis
- SHAP Explainability
- Feature Reduction Optimization

Results:
| Metric | Result |
|---|---|
| Accuracy Before GA | 97.36% |
| Accuracy After GA | 98.24% |
| Feature Reduction | 30 → 11 |
| ROC-AUC | 0.98 |

Dataset:
https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data

Technologies Used:
- Python
- Scikit-learn
- DEAP
- SHAP
- Google Colab

How to Run:
1. Clone repository
2. Install requirements
3. Run notebook
